# Startup Hunter

Goal: Every day find startups Jeff should contact.

Search only these categories:

- AI Wearables
- AI Hardware
- Robotics
- Healthcare Devices
- Consumer Electronics

Include only startups founded in the last 24 months.

Reject companies that are clearly too large, too old, software-only, or unlikely to need hardware/ODM help.

Rank by:

Hardware Probability x ODM Probability x Contact Urgency

Output only the best matches.
